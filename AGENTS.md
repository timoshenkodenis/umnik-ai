# AGENTS.md

## Project
Это проект Умник AI — детский учебный AI-помощник.

## Working directory
Актуальное приложение находится только в папке:
umnic/

Рабочие файлы:
- umnic/index.html
- umnic/api/chat.js
- umnic/assets/

## Do not use
Не использовать старые ZIP-архивы, старые версии и случайные папки.

## Secrets
Не коммитить API-ключи.
OPENROUTER_API_KEY должен храниться только в Vercel Environment Variables.
OPENROUTER_MODEL тоже хранить в Vercel Environment Variables.

## Deployment
Проект деплоится через Vercel.
Root Directory: umnic

## Current features
Перед любыми изменениями учитывать, что уже есть:
- стартовый экран;
- кнопка «Начать чат»;
- чат;
- Free/Plus;
- AI/демо режим;
- голосовой звонок;
- авто-слушание;
- фото внутри звонка;
- остановка речи кнопкой;
- остановка речи голосом;
- профиль ученика;
- память Умника;
- меню профиля;
- OpenRouter API через api/chat.js.

## Rules
1. Не менять визуальный стиль без отдельной просьбы.
2. Не ломать существующие функции.
3. Для крупных изменений создавать отдельную ветку.
4. После изменений проверять стартовый экран, чат, Plus, звонок, фото, профиль и память.
5. Если есть риск сломать проект — сначала предложить план.
