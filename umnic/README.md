# Умник AI — Vercel версия

Эта версия сделана специально для деплоя по ссылке через GitHub + Vercel.
Локально запускать `npm install` не нужно.

## Файлы

- `index.html` — интерфейс приложения.
- `api/chat.js` — серверная функция, которая обращается к OpenAI.
- `assets/` — картинки Умника и логотип.
- `.env.example` — пример переменных окружения.

## Как задеплоить через GitHub + Vercel

1. Создай новый репозиторий на GitHub.
2. Загрузи туда все файлы из этой папки.
3. В Vercel нажми Add New Project / Import Project.
4. Выбери этот GitHub-репозиторий.
5. В настройках проекта добавь Environment Variable:

```env
OPENAI_API_KEY=твой_новый_api_ключ
```

Дополнительно можно добавить:

```env
OPENAI_MODEL=gpt-4.1-mini
```

6. Нажми Deploy.

После деплоя приложение будет открываться по ссылке Vercel.

## Важно по безопасности

Не вставляй API-ключ в `index.html` или другие файлы фронтенда.
Ключ должен храниться только в Environment Variables на Vercel.

Ключ, который был отправлен в чат, лучше удалить в OpenAI Platform и создать новый.
