# Умник AI — Vercel + OpenRouter версия

## Что нужно добавить в Vercel

Settings → Environment Variables:

```env
OPENROUTER_API_KEY=твой_ключ_OpenRouter
OPENROUTER_MODEL=google/gemini-3.1-flash-lite
```

После добавления переменных нужно сделать новый Deploy / Redeploy.

## Файлы

- `index.html` — интерфейс приложения.
- `api/chat.js` — серверная функция Vercel, которая отправляет запрос в OpenRouter.
- `assets/` — картинки и логотип.


## Voice Call Mode

В этой версии добавлен режим «Созвон с Умником».

Условия показа кнопки:
- тариф: Умник Плюс;
- режим ответа: AI;
- экран чата.

Для работы AI в Vercel должны быть переменные:
- OPENROUTER_API_KEY
- OPENROUTER_MODEL, например google/gemini-3.1-flash-lite

Голосовой режим лучше всего работает в Google Chrome.


## Улучшение голоса

В этой версии голос Умника стал мягче:
- выбирается лучший русский голос из доступных в браузере;
- речь чуть медленнее;
- тон чуть выше;
- длинные ответы разбиваются на короткие фразы с паузами.

Важно: Web Speech API использует голоса браузера/устройства. Максимально живой человеческий голос потребует отдельный TTS-сервис.


## Voice Call V3: фото внутри звонка + авто-слушание

Что изменено:
- фото добавляется прямо в окне звонка;
- снимок отображается в чате звонка;
- после фото можно голосом спросить про него;
- Умник отправляет в AI и историю разговора, и фото;
- после входа в звонок микрофон запускается автоматически;
- когда Умник думает или говорит, речь пользователя не учитывается;
- после ответа Умника микрофон снова включается автоматически.

Важно:
- Авто-слушание лучше всего работает в Google Chrome.
- При первом запуске нужно разрешить доступ к микрофону и камере.


## Voice Call V5: остановка ответа

Добавлено:
- кнопка «Продолжить» теперь работает как умная кнопка:
  - когда Умник говорит: «Остановить ответ»;
  - когда пользователь говорит: «Завершить фразу»;
  - когда пауза: «Продолжить».
- во время ответа Умника можно сказать:
  - «Умник, перестань говорить»
  - «Умник, стоп»
  - «Хватит»
  - «Стоп»
- Умник остановит озвучку, текст ответа останется на экране, затем микрофон снова начнёт слушать новый вопрос.

Важно: голосовая остановка лучше работает в Chrome и особенно хорошо — если использовать наушники, чтобы микрофон не ловил голос самого Умника.


## V6: Профиль ученика + память

Добавлено:
- экран «Профиль ученика» в настройках;
- сохранение имени, возраста, класса, сложного предмета, сложных тем и стиля объяснения;
- память Умника в localStorage;
- Умник запоминает частые темы, сложные темы, голосовые вопросы и фото;
- профиль и память отправляются в AI;
- Умник начинает подстраивать ответы под ученика.

Важно:
- память хранится только в браузере на устройстве;
- после регистрации в будущей версии память можно будет синхронизировать через базу данных.
