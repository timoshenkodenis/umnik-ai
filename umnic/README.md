# Умник AI — Vercel версия v2

Что изменено в v2:

- Умник в чате перенесён в верхнюю шапку между логотипом и настройками.
- На главном экране убрана кнопка настроек, логотип расположен по центру.
- Кнопка настроек заменена на шестерёнку.
- Поле ввода, скрепка, микрофон и отправка приведены к одному размеру.
- Иконки микрофона, скрепки и отправки заменены на более аккуратные SVG.
- Шрифт сообщений стал легче, чтобы на экране помещалось больше текста.
- Добавлена проверка имени и возраста на стартовом экране.
- Чаты больше не сохраняются после перезапуска страницы: каждый запуск начинается с нуля.
- Строку чатов можно листать перетаскиванием.
- Добавлен режим редактирования чатов: удаление и перемещение.

## Как обновить текущий GitHub-репозиторий

1. Распакуй этот архив.
2. Открой GitHub → репозиторий `umnik-ai` → папку `umnic`.
3. Нажми `Add file` → `Upload files`.
4. Перетащи содержимое распакованной папки: `index.html`, `api`, `assets`, `package.json`, `README.md`.
5. Нажми `Commit changes`.
6. Vercel сам сделает новый деплой.

Если деплой сам не начался: Vercel → Deployments → Redeploy.
